### 自定义值

“自定义属性”的作用是在CSS中创建变量。

```
html{
	--base-color:#639;
	--highlight-color:#EAE;
}
h1{
	color:var(--base-color)
}
h1{
	color:var(--highlight-color)
}
```

自定义标识符以两个连字符开头(--)，调用的方法是使用var()值类型，这些名称是区分大小写的。

自定义属性的特性：作用域在一定的范围内。、

```
html{
	--base-color:#639;
}
aside{
	--base-color:#F60;
}
h1{
	color:var(--base-color)
}
#  不讲究注释规范
<body>
	<h1>h1</h1> #color: #639;
	<aside>
		<h1>h1</h1> # color: #F60;
	<aside/>
	<h1>h1</h1> #color: #639;
</body>
```
